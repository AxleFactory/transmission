# contact-texter-android

The Android version of the Contact Texter.

### Installation

1. Clone the project
  ```
  git clone git@github.com:Bernie-2016/contact-texter-android.git
  ```

2. Build the project
  ```
  cd context-texter-android
  npm install
  ```

3. Run the project
  ```
  react-native run-android
  ```
